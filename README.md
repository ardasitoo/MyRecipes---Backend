# MyRecipes---Backend

MyRecipes ist eine Web-Anwendung zur digitalen Verwaltung von Rezepten. Nutzer können Rezepte erstellen, anzeigen, bearbeiten und löschen. Jedes Rezept enthält Informationen wie Titel, Beschreibung, Zubereitungszeit sowie eine Liste von Zutaten und Zubereitungsschritten. Zusätzlich können Rezepte in Kategorien eingeordnet werden, um eine bessere Übersicht und Organisation zu ermöglichen. Ziel der Anwendung ist es, Rezepte strukturiert zu speichern und einfach zugänglich zu machen.
